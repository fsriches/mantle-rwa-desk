    how it works

    Four layers, one run.

    The tool is small on purpose. It runs entirely in your browser and points at any Mantle or crypto asset you type in. A terminal version exists too, for anyone who prefers the command line.

    01 · FETCH

    Sources

    Price from CoinGecko and DefiLlama, TVL from DefiLlama. Each source is a named function. Add one, add a function.

    02 · RECONCILE

    Compare

    Same metric, many feeds. It measures the spread and shows every source by name. Agreement is verified, disagreement is flagged.

    03 · SIGNALS

    Flag

    Readable rules: sources disagree, weekly drawdown, distance from ATH, thin price confidence. Each flag names the number behind it.

    04 · BRIEF

    Read

    A short readout you can screenshot. Numbers come from the feeds, not from a model, so nothing is invented.
